Security

Encrypted before it
touches the disk.

Every clip is wrapped in hybrid post-quantum encryption. Even if someone copies your database, they get nothing.

1

You copy something

Clipboard Vault captures the content and immediately encrypts it with a fresh ephemeral key pair.

2

Hybrid key exchange

X25519 and ML-KEM-768 run in parallel. Their shared secrets are combined via HKDF to derive a unique AES-256 key.

3

AES-256-GCM seals it

Content is authenticated-encrypted. The ciphertext, nonce, and ephemeral keys are stored. Plaintext is never written.

4

Touch ID to decrypt

Private keys live in macOS Keychain. Only your fingerprint or password can unlock them. Keys are zeroed from memory on lock.
